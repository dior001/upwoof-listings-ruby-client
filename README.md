# UpWoof Listings Ruby Client

A Ruby wrapper for the UpWoof Listings API.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'upwoof_listings'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install upwoof_listings

## Documentation
